# CC6004 — Multi-Layer Dynamic Shift Cipher (MLDSC)

## Overview

This project presents the design and implementation of a custom educational cipher: the Multi-Layer Dynamic Shift Cipher (MLDSC).

The system combines multiple transformation layers and a SHA-256-based key schedule to demonstrate core cryptographic principles.

## Components

- Python implementation of the cipher
- HTML-based interactive demo
- Supporting academic documentation

## Status

Completed as part of CC6004 Network and Cloud Security module.
